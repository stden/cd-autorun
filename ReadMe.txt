1. Если автоматического запуска системы не произошло, запустите файл Autorun.exe из корневого каталога
2. В оболочке "Журнала в журнале" есть возможность установки его на компьютер: Меню -> Установить
3. Для удаления "Журнала в журнале" с компьютера необходимо выбрать Меню -> Удалить.
!ВНИМАНИЕ: для удаления "Журнала в журнале" с локального компьютера необходимо проделать операцию удаления используя диск, вставленный в дисковод

4. Все модели используещиеся в "Журнале в журнале" собраны в каталоге \data
5. Все статьи сопровождающие модели собраны в каталоге \PDF
6. В каталоге \msi собраны программы, необходимые для работы моделей собранных в "Журнале в журнале"

Если Вы заметили ошибку в работе "Журнала в журнале" или у Вас есть предложения, пишите нам в редакцию - kio@ipo.spb.ru.
Мы непременно ответим!
